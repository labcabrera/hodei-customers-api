= Hodei Customers API

http://localhost:8080/graphiql

Ejemplo de consulta:

----
{
  customer(id : "5f9c5500e930620621202e89") {
    id
    idCard {
      type
      number
    }
    name
    surname1
    surname2
    birth
    birthCountry {
      id
      name
    }
    fiscalAddress {
      country {
        id
        name
      }
      province {
        id
        name
      }
      locality
      zipCode
      road
    }
    professionInfo {
      profession {
        id
        name
      }
    }
  }
}
----
